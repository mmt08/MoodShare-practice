Аннотация проекта

```

MoodShare — мобильное приложение для ежедневного отслеживания эмоционального состояния пользователя. Основная цель проекта заключается в развитии эмоциональной осознанности, формировании полезных привычек самонаблюдения и профилактике эмоционального выгорания.

Пользователь может фиксировать своё настроение, добавлять заметки, анализировать статистику и получать персонализированные напоминания.

Основные возможности

  • Фиксация настроения одним нажатием.
  • Добавление заметок к записям.
  • Статистика и графики настроения.
  • Система напоминаний.
  • Локальное хранение данных.
```
